Overview

File format information for some ZeroEngine formats.

Zero CRC

The CRC used in .msh files is 4 bytes long and is stored in pure hex. The algorithm is proprietary (Zero because it's used in Pandemic's Zero Engine). An example implementation of the Zero CRC can be found here (python).

ATRB Calculator

Use this to calculate flag values, convert render type and data values to hex and convert from and to the .msh ATRB hex chunk.

Note that Single-sided and Double-sided Transparency are exclusive, so selecting both might not have the desired/any effect. Check out the ATRB chunk for more information on each flag.

unsigned __cdecl CalcLowerCRC(const char* str, unsigned crc)
{
    const unsigned char *p;

    crc = ~crc;
    if (str) {
        for (p = (const unsigned char *) str; *p; p++) {
            crc = (crc << 8) ^ table32[(crc >> 24) ^ toLower[*p]];
        }
    }
    return ~crc;
}
