    Render Types Information


    RENDER GLOW
    #1 01
    Description: Render Glow
    Data0: None
    Data1: None
    Additional: None

    RENDER DETAIL
    #2 02
    Description: Render Detail. On GT this is mixed up with Render Lightmap.
    Data0: Repetition along U.
    Data1: Repetition along V.
    Additional: Texture(detail texture).

    RENDER SCROLLING
    #3 03
    Description: Render Scrolling. Scrolls the UV values along the texture.
    Data0: U(Horizontal) Scroll Speed.
    Data1: V(Vertical) Scroll Speed.
    Additional: None

    RENDER SPECULAR
    #4 04
    Description: Render Specular. Enables Specular. Specular Flag doesn't seem to be needed for it to work. Specular power is affected by material color and Specular Decay(gloss).
    Data0: None
    Data1: None
    Additional: None.

    RENDER GLOSS MAP
    #5 05
    Description: Render Gloss Map(Specular Map). This uses the diffuse texture's alpha channel for specular intensity.
    Data0: None
    Data1: None
    Additional: Specular Color and Decay are used to determine specularity. Make sure the values are valid. If they are too low the specular might not appear. If the diffuse texture doesn't have an alpha channel specular will be turned off.

    See also: Specular discussion on GT

    RENDER CHROME
    #6 06
    Description: Render Chrome. Environment map.
    Data0: None
    Data1: None
    Additional: Additional texture(environment cube map). If this texure is not set the local envmap of the map will be used.

    RENDER ANIMATED
    #7 07
    Description: Render Animated. Animates UVs to loop through different UV positions on the texture. UVs should be mapped to the first cell and not the whole texture. Cells have to be square and will be calculated automatically from the number of frames.
    Data0: Number of frames in the texture.(Minimum: 4)
    Data1: Animation Speed.
    Additional: None
    See also: Render Animated discussion on GT.

    #22 16

    Data0: None
    Data1: None.

    RENDER BUMPMAP
    #27 1b
    Description: Bump or normal map. Material color and specular decay affect the highlight ingame.
    Data0: None
    Data1: None.
    Additional: Texture(greyscale bumpmap or normalmap). The bumpmap needs a .option with -format bump in it. Specular flag should be enabled for better effect.
    See also: Bumpmap tutorial on GT


    RENDER BUMPMAP + GLOSS MAP
    #28 1c
    Description: Bump or normal map with specular map.
    Data0: None
    Data1: None.
    Additional: Texture(greyscale bumpmap or normalmap). The bumpmap needs a .option with -format bump in it. Optional texture which controls the specular intensity through it's alpha channel. Specular flag should be enabled.
    See also: Bumpmap tutorial on GT
