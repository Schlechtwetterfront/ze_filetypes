    General Information

    CRCs are used in the animation chunks of .msh files. The algorithm used to calculate them is as of now unknown. However there's a tool in SWBF2_ModTools called drcrc.exe, which is able to calculate those CRCs. Other tools like ZenAsset.exe(munges files, eg animations) are most likely able to calculate and reverse this special sort of CRC(labelled 'Zero'). The functions doing this aren't exposed to the commandline though. The CRC used in .msh files is 4 bytes long and is stored in pure hex.

    Resources

    • drcrc.exe Calculates a Zero CRC from the current content of the clipboard. The clipboard will contain case %crc%: // "%string_used_to_calc_crc%".
    • wDrCRC.exe A wrapper for drcrc created by tirpider. This has some additional functionality like comparing to existing CRCs from the game assets to help getting the string from a given crc. You're able to compare a given crc to a string, too.
